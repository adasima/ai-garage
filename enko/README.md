# enko (Omoshiro.dom)

語呂合わせやワードハックを駆使して、クリエイティブなドメイン名や単語の組み合わせを生成する次世代ジェネレーターです。

## プロジェクトの概要

`enko` は、入力されたキーワードから「インパクトがあり、覚えやすい」ドメイン名や単語のアイデアを提案します。単なる文字列の結合ではなく、日本語の語呂合わせ（goroawase）や、ドメイン末尾を単語の一部に見立てる「ワードハック（wordhacks）」を活用するのが特徴です。

## 主な機能

- **⚡ Voltage システム**: 生成されたアイデアの「キレ」をスコアリング。レベルが高いほど、より印象的なドメインです。
- **🔢 語呂合わせ生成**: 日本語の読みを数字に変換し、覚えやすいドメインを提案します。
- **✂️ ワードハック**: `en.ko` や `ma.ki` のように、TLDを単語の一部として利用するアイデアを生成します。
- **🧠 関連アソシエーション**: 入力ワードに関連するミームやトピックからもアイデアを広げます。
- **🏹 モード切替**:
    - **Hunter Mode**: 厳選された、実用性の高いアイデアを表示。
    - **Museum Mode**: 取得済みのドメインも含め、インスピレーションのための全アイデアを表示。

## 技術スタック

- **Core**: [Next.js 16 (App Router)](https://nextjs.org)
- **Runtime**: React 19
- **Logic**: 
    - [Wanakana](https://wanakana.com) (ローマ字・ひらがな変換)
    - カスタム語呂合わせ・ワードハックエンジン
- **Styling**: Tailwind CSS 4, Framer Motion (アニメーション)
- **Backend**: Supabase (状態管理・データ永続化)

## 開発を始める

### 1. セットアップ

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

### 3. ロジックのデバッグ

プロジェクトルートにある以下のスクリプトでロジックの動作を確認できます。

```bash
npx tsx debug-japanese.ts
npx tsx debug-unkasu.ts
```

## ディレクトリ構造

- `src/lib/domain-logic/`: ドメイン生成、語呂合わせ、ワードハックのコアロジック。
- `src/components/`: UIコンポーネント（`HeroInput`, `ResultStream`など）。
- `src/app/`: ページのルーティングとレイアウト。
